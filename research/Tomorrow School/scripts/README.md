# Scripts

Скрипты для исследования платформы Tomorrow School.

## Использование
Все JS-скрипты предназначены для запуска через **Chrome DevTools → Sources → Snippets** (не через консоль, т.к. там обрезаются длинные строки).

| Файл | Описание | Как использовать |
|------|----------|-----------------|
| `application-dump.js` | Полный дамп Application данных | Запустить на странице аудита |
| `auto-fill-audit.js` | Авто-заполнение и отправка аудита | На странице с формой аудита |
| `websocket-interceptor.js` | Перехват WebSocket сообщений | Запустить перед открытием аудита |
| `bruteforce-codes.js` | Брутфорс кодов верификации | Запустить в браузере или node.js |
